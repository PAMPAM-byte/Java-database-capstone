# Doctor User Stories
- As a Doctor, I want to log in securely so that I can access my dashboard.
- As a Doctor, I want to view my schedule so that I can manage my appointments.
- As a Doctor, I want to approve or decline appointments so that I can manage my workload.

# Patient User Stories
- As a Patient, I want to register and log in so that I can book appointments.
- As a Patient, I want to search doctors by specialty and availability.
- As a Patient, I want to book and cancel appointments easily.

# Admin User Stories
- As an Admin, I want to log in securely so that I can manage the system.
- As an Admin, I want to add, update, or remove doctors.
- As an Admin, I want to monitor system usage and performance.
